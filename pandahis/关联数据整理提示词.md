## 关系层级规则

关系图谱采用树状层级结构，所有关系均按"主体 → 一级 → 二级 → 三级 → 四级"组织。

### 一级关系

一级关系是与史略主题直接建立关系的对象。

例如：
- 人物：父母、妻子、子女、老师、弟子、君臣、政敌、盟友等。
- 事件：主要参与者、发动者、对立方、发生地点等。
- 制度：创立者、执行机构、改革者等。
- 民俗：起源人物、起源地区、代表节日等。
- 论著：作者、编者、核心思想、代表人物等。

### 二级至四级关系

只有具有**连续传承、继承或隶属关系**的节点，才允许继续向下展开。

允许继续展开的关系包括（但不限于）：

- 家庭血缘（父子、祖孙等）
- 师承关系（老师→弟子）
- 思想传承（提出→继承→发展）
- 制度演变（前身→改革→继承）
- 著作关系（作者→注释→续作）
- 组织隶属（机构→下属机构）

每一级均表示上一节点的直接关联对象，而不是始终与主题直接关联。

例如：

黄帝
→ 嫘祖（一级）
→ 玄嚣（二级）
→ 蟜极（三级）
→ 帝喾（四级）

孔子
→ 曾参（一级）
→ 子思（二级）
→ 孟子（三级）

科举制
→ 察举制（前身，一级）
→ 九品中正制（二级）
→ 隋唐科举（三级）
→ 明清科举（四级）

### 不继续展开的关系

以下关系原则上只保留一级节点，不再继续展开：

- 君臣
- 同僚
- 敌对
- 联盟
- 战争参与
- 外交关系
- 地域关系
- 一般历史关联

除非继续展开能够直接帮助理解当前主题，否则不得无限递归。

### 停止展开规则

满足任意一项，即停止继续展开：

1. 已达到四级。
2. 后续节点已明显脱离当前史略主题。
3. 后续节点已有独立史略，可由其自身关系图继续展开。
4. 后续关系仅属于一般历史关联，不属于传承或隶属关系。
5. 继续展开不能增加对当前主题的理解。

### 完整性要求

关系图谱应遵循"宁可完整，不可无限扩张"原则。

对于每个关系类别，应优先保证一级关系完整；仅当存在明确的传承、继承或隶属关系时，再逐级展开。不得为了增加层级而机械扩展，也不得遗漏对理解主题具有重要意义的关键关系。
